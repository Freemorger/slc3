# Simple Local Chat v3 
Rewriting completely on Go! 
Note: Currently indev

# How to run
## Dependecies
Latest Go compiler
## Running slc3
Download repo:
`git clone https://github.com/Freemorger/slc3.git`
Go to repo dir:
`cd slc3`
Run server:
`go run server.go`
And/or run client:
`go run client.go`
